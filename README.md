# Carousel-chart

#### 介绍
轮播图：自动播放+手动点击播放
#### 分析
①：右侧按钮点击，变量++，如果大于等于8，则复原0

②：左侧按钮点击，变量--，如果小于0，则复原最后一张

③：鼠标经过暂停定时器

④：鼠标离开开启定时器
